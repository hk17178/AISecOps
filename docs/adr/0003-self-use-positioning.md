# ADR-0003 · 项目定位为"自建自用内部工具"

- 状态：Accepted
- 日期：2026-05-26
- 决策者：muzi

## 背景

初版 PRD / 技术设计 / 开发计划按"商业化 AISecOps 平台"定位起草，对标 CrowdStrike Charlotte / Microsoft Security Copilot / 奇安信 QAX-GPT，规划了双部署模式（SaaS + 本地）、多租户、6 角色画像、商业化路径、ISO 27001 / SOC2 / 等保 2.0 认证、24 Sprint / 42 周到 GA。

实际定位经确认是 **"自建自用 · 自己小公司/部门 10-100 人内部工具"**，不是商业产品。继续按商业化基线推进会：
- 过度规划多租户 / License / 售前 / MSSP，**做出来没人用**
- 追合规认证耗费大量时间，**自用不需要**
- 24 Sprint 长跑节奏，**单人节奏耗不动**
- 用"赢竞品"思维写文档，**写得累 + 看着也累**

## 决策

将项目定位明确为**自建自用内部工具**，并据此**瘦身**所有 P0 设计文档：

### 砍掉 / 简化的方面

| 项 | 处置 |
|---|---|
| 双部署模式（SaaS + 本地） | 仅保留**单一本地部署**（在你环境跑） |
| 多租户 + 客户隔离 | **删除**，单租户 + 内部 RBAC |
| 6 角色画像（MSSP / 合规官 / SOC 总监等） | 简化为 **3 个内部角色**：管理员 / 分析师 / 普通查看 |
| 商业化路径（首单 / MSSP / License / 出海） | **全部删除** |
| ISO 27001 / SOC2 / 等保 2.0 认证 | **不追认证**；借鉴控制项思路提升安全度即可 |
| 客户兼容性承诺书 | **删除** |
| 性能指标 EPS 50K | 降级到 **5K EPS 内**（实际可能 500-2000） |
| 团队演进路线（销售 / 售前 / 实施） | **删除**；保留"自己 + 几个同事帮忙"小团队节奏 |
| DSLM 飞轮 | **P3+ 视数据量评估**，初期不做（自用数据量大概率不够） |
| 24 Sprint / 42 周到 GA | 改为 **6-8 Sprint MVP** |

### 保留 / 强化的方面

| 项 | 为何保留 |
|---|---|
| 12 层架构 | 单人长期维护必须分层清晰 |
| 34 条铁律（CONSTRAINTS） | AI 自动化误判同样会让你封错 IP、删错数据 |
| 抑制幻觉 C-24/26/27 | 自用更容易"信 AI"，没有团队 review 兜底 |
| HITL（C-8） | 自用没有第二双眼睛，HITL 是唯一安全网 |
| MCP-first（L06） | 接入工具复用机制，自用更需要 |
| 业界对标 | 作为**学习参考**保留，不为了 1:1 跟进 |
| ADR 流程 | 三个月后你想不起为什么这么选 |
| 中文注释 + docstring（C-28） | 未来唯一读者就是你 |

### 新增 / 强化的方面（自用特有）

| 项 | 为什么 |
|---|---|
| LLM token 成本控制 | 自用钱包自己出，eval 跑太勤会破产 |
| 学习价值最大化 | 自用项目同时是学习项目，**架构清晰 + 接触新技术 > 功能全** |
| 重启手册 | 哪天不维护了，文档要让自己 5 分钟内重新上手 |
| 内部团队体验 | 同事用得不爽就不用了，UI/响应 > 功能多 |

## 后果

**正面**：
- 文档量大幅减少，**写得动 + 看得动**
- 开发节奏务实，MVP 几个月可见
- 避免在"卖给谁"的伪命题上空转
- 工程约束（铁律 / HITL / 12 层）反而**更聚焦在真正重要的地方**

**负面**：
- 如未来想商业化，需要补回多租户 / 合规 / 双模式（**预计 3-6 个月增量**）
- 业界对标变成"学习"而非"赢"，对外展示价值降低

**风险**：
- 真有一天想商业化时，自用版本可能架构选择不利于扩展 → 通过保留 12 层架构 + ADR 流程缓解，单租户改多租户的成本可控

## 备选方案

### 备选 1：维持商业化定位
- 拒绝理由：与实际不符，会持续做无用功

### 备选 2：保留商业化路径但标"V2 再做"
- 拒绝理由：文档充斥"V2 可能要做"会污染当前判断；需要时再写 ADR 补就好

### 备选 3：完全推翻重写
- 拒绝理由：现有结构（12 层 / 铁律 / harness / 架构 V2）大部分仍适用，**瘦身而非推翻**
